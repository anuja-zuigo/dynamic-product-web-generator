# Dynamic Product Web Generator

An enterprise-inspired Product Management System that enables administrators to bulk manage products using Excel spreadsheets. The application processes CRUD operations from Excel files and dynamically generates product web pages without requiring code changes or application downtime.

---

## 📌 Problem Statement

**Dynamic Web Page Generation of Products Using Excel Row Description**

---

## 🎯 Project Objective

Build a web application that allows an administrator to:

- Upload an Excel sheet containing product information.
- Perform Create, Update and Delete (CRUD) operations using Excel.
- Upload product images.
- Dynamically generate product pages.
- Manage products without modifying application code.
- Demonstrate a basic AI chatbot for product-related queries.
